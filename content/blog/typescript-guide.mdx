# TypeScript ile Tipe Güvenli Kod Yazma

TypeScript, JavaScript'e statik tipleme katarak kodunuzun daha güvenli ve bakımı kolay olmasını sağlar.

## Neden TypeScript?

- **Type Safety**: Compile time'da hataları yakala
- **Better IDE Support**: Daha iyi autocomplete ve refactoring
- **Self Documenting**: Kodun ne beklediği açık
- **Production Ready**: Büyük projelerde tercih ediliyor

## Basit Örnek

```typescript
interface User {
  id: number;
  name: string;
  email: string;
}

function createUser(user: User): void {
  console.log(`Created user: ${user.name}`);
}

// ✅ Doğru
createUser({ id: 1, name: "John", email: "john@example.com" });

// ❌ Hata - email gerekli
createUser({ id: 2, name: "Jane" });
```

## Sonuç

TypeScript ile daha güvenli, daha kolay okuyabilen ve daha kolay bakım yapılabilen kod yazabilirsiniz.
